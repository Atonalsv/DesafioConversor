1-DESCRIPCION DEL PROYECTO:

DESAFIO CONVERSOR DE MONEDAS es una aplicacion JAVA que cumple la funcion de conversion de divisas.

***************************************************************************************************

2- ESTADO DEL PROYECTO: Finalizado.

**************************************************************************************************
3- FUNCIONALIDADES:
El proyecto permite al usuario seleccionar una divisa, para luego hacer una conversion a un tipo
de moneda diferente mediante la utilizacion de un menu interactivo.

***************************************************************************************************

4- GUIA DE USUARIO:

-Abra la aplicacion.
-Lea el menu y elija la opcion valida proporcionada en la lista.
-Ingrese la cantidad monetaria en la divisa seleccionada que quiera convertir.
-El resultado sera actualizado en pantalla.
-El programa luego le mostrara las opciones disponibles en caso el usuario quisiera continuar.
-De no querer continuar o querer salir de la aplicacion, seleccione opcion 7 para salir.
